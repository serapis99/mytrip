# Atributos

los atributos son valores que configuran los elementos o ajustan su comportamiento. 

En terminos generales hay dos tipos de atributos.

comunes:
```HTML
<p atributo="valor">
```
booleanos:
```HTML
<p atributo>
```

Ejemplos

```HTML
<!--Comun-->
<html lang="es-CO">
<meta charset="UTF-8">

<!--booleanos-->
<a href="https//www.google.com" download>
```
